Experiment Cookbook: A Tutorial on oTree Programming with Austėja Kažemekaitytė Book

oTree is a Python-based framework for developing interactive experiments for behavioral and experimental economics. In this online book, we provide a tutorial on oTree programming as well as a collection of recipes for common tasks and problems.

The book also includes introductions to Python, HTML, CSS, and JavaScript for those unfamiliar with these languages.
